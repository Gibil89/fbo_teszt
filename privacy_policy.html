Adatvédelmi irányelvek

Utolsó frissítés: 2025. május 8.

Ez az alkalmazás ([FBŐ Vizsgafelkészítő]) nem gyűjt, nem tárol és nem továbbít semmilyen személyes adatot vagy felhasználói információt.
Az alkalmazás kizárólag a működéséhez szükséges adatokat kezeli, és minden funkciója megfelel a vonatkozó adatvédelmi előírásoknak.

Külső szolgáltatások

  • Firebase (Google): Az alkalmazás kérdésadatbázist és adatfrissítést a Firebase szolgáltatáson keresztül végez. A Firebase adatkezeléséről részletesen itt olvashat: Firebase adatvédelmi irányelvek.
  • YouTube: Az alkalmazásban beágyazott, nem nyilvános (unlisted) YouTube videók találhatók. A YouTube adatkezelési gyakorlatáról itt talál információt: YouTube adatvédelmi irányelvek.
  • GitHub Pages: Az adatvédelmi irányelvek oldal a GitHub Pages szolgáltatásán keresztül érhető el. A GitHub adatkezelési szabályairól itt olvashat: GitHub adatvédelmi irányelvek.
